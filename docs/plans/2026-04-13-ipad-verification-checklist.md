# iPad 真机验收清单

> 适用范围：Flutter iPad 真机验收  
> 目标：验证移动端主流程、适老化模式、消息中心、OCR 计划、AI 展示和药箱/家庭体验。

## 1. 启动前确认

- [ ] iPad 已连接 Mac
- [ ] iPad 已解锁
- [ ] iPad 已信任此电脑
- [ ] Mac 已安装完整 Xcode
- [ ] Mac 已安装 Flutter SDK
- [ ] `flutter doctor` 无阻塞错误
- [ ] `flutter devices` 能看到 iPad
- [ ] 后端服务已启动
- [ ] Web 服务已启动
- [ ] 测试账号已准备好

## 2. 登录验证

- [ ] 能打开登录页
- [ ] 能登录
- [ ] 登录后能进入首页
- [ ] 退出后能回到登录页

## 3. 首页验证

- [ ] 底部导航能切换
- [ ] 概览页能显示家庭数据
- [ ] 今日待服统计可见
- [ ] 临期提醒可见
- [ ] 低库存提醒可见
- [ ] 今日服药时间轴可见

## 4. 适老化模式验证

- [ ] 能进入老人专用模式
- [ ] 字号足够大
- [ ] 按钮足够明显
- [ ] 今日服药入口足够突出
- [ ] 单步打卡可完成
- [ ] 小屏和横屏不溢出

## 5. 服药模块验证

- [ ] 今日服药列表能打开
- [ ] 计划列表能打开
- [ ] 统计页能打开
- [ ] 打卡成功
- [ ] 跳过成功
- [ ] 状态刷新正确

## 6. 药箱模块验证

- [ ] 药品列表能打开
- [ ] 药品详情能打开
- [ ] 库存状态显示正常
- [ ] 拍照识别入口可用
- [ ] 手动新增入口可用

## 7. 消息中心验证

- [ ] 能看到站内消息列表
- [ ] 能看到未读数
- [ ] 可标记已读
- [ ] 可全部已读

## 8. OCR 辅助计划验证

- [ ] OCR 结果页能打开
- [ ] 识别结果可编辑
- [ ] 可生成计划草稿
- [ ] 提交前能确认

## 9. AI 模块验证

- [ ] 文本问诊能发送
- [ ] 图片分析能发送
- [ ] 结构化结果能显示
- [ ] 咨询历史可打开

## 10. 家庭与病历验证

- [ ] 家庭可切换
- [ ] 成员可切换
- [ ] 健康档案可查看
- [ ] 健康档案可编辑
- [ ] 病历列表可查看
- [ ] 病历详情可查看

## 11. 需要记录的问题

- [ ] 页面截断
- [ ] 字体过小
- [ ] 按钮太密
- [ ] 点击区域过小
- [ ] 数据加载慢
- [ ] 页面切换卡顿
- [ ] 文案不清楚
- [ ] 接口失败

## 12. 验收结论

- [ ] 通过
